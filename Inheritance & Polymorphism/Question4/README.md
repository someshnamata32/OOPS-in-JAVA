4. Develop an online shopping system with hierarchical inheritance. Create a base class
Product and two derived classes Electronics and Clothing. Each class should have
specific properties and methods. You can use constructor or setter methods to
initialize the data members.
Data Members:
• Product: productName (String), price (double)
• Electronics: brand (String), warrantyPeriod (int)
• Clothing: size (String), material (String)
Member Functions:
• Product: displayDetails()
• Electronics: displayDetails()
• Clothing: displayDetails()
