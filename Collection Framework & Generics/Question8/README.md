8. You are designing a triage system for a hospital's emergency room, where patients are
prioritized based on the severity of their condition. Define a Patient class with properties
name and severity (integer, where a lower number means more severe). Create a
PriorityQueue to store patients, ordered by severity. Add 5 patients to the queue. Display
the patients in order of severity (highest severity first). Remove the most critical patient
and display the remaining patients.
